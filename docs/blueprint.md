# **App Name**: Saucy's Southern BBQ & Cuisine

## Core Features:

- Menu Display: Dynamically display the restaurant's menu, including item descriptions and prices.
- Order Placement: Enable users to place orders directly through the webpage.
- Chatbot Assistance: Implement a chatbot to assist users with menu inquiries, order status, and general information, using an AI tool to decide when the bot's guidance can be most helpful.
- Restaurant Locator: Display a map of the restaurant location.
- Special Offers: Highlight any ongoing promotions or special offers.
- Jobs Tab: Provide information about job openings and application process within the same page.
- Catering Tab: Provide information and ordering options for catering services within the same page.

## Style Guidelines:

- Primary color: Warm terracotta (#E07A5F) evoking comfort and appetite, balanced with black (#000000) accents to align with the logo.
- Background color: Light beige (#F8F1ED) to create a welcoming and clean environment.
- Accent color: Mustard yellow (#F2CC8F) for highlights and interactive elements, complementing the logo's gold tones.
- Headline font: 'Playfair', a modern serif that exudes elegance and clarity.
- Body font: 'PT Sans', a humanist sans-serif for a contemporary feel. Used to complement Playfair in readability.
- Use minimalist icons to represent menu categories, options, jobs, and catering.
- Implement a clean, grid-based layout for easy navigation within a single page, using tabs or sections to access different content (menu, ordering, location, jobs, catering).
